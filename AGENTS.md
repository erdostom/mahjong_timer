this is a mahjong timer project

the UI should be very simple, a timer counting down in large legible font, showing minutes and seconds. small buttons for start/stop/continue and reset
when the timer reaches zero, the background should change to a red color and the timer should start counting up.
another small button in the corner for settings

settings:
- total time (minutes) 
- play sound at (minutes remaining)
- play sound at 0

architecture:
this should be a react app built with vite
